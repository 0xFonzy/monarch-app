## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- Yarn package manager

### Installation

1. Clone the repository
2. Install dependencies:

```bash
yarn
```

### Development

Run the development server:

```bash
yarn dev
```

The app will be available at `http://localhost:5173`
